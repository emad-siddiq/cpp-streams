To build Makefile:
```
rm -rf build && mkdir build
cmake .. 
make
```

To build for Xcode, cd into project folder and:
```
rm -rf build && mkdir build
cd build
cmake .. -G Xcode
open `<project_name>.xcodeproj`
```
